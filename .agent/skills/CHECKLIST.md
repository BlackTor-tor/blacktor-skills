# New Skill Checklist

Before pushing a new skill into this repository, confirm:

- [ ] Correct category selected
- [ ] Path follows `.agent/skills/<category>/<skill-name>/`
- [ ] Skill name uses lowercase kebab-case
- [ ] `SKILL.md` exists for full skills
- [ ] Scripts are under `scripts/`
- [ ] Supporting docs are under `references/`
- [ ] No secrets, tokens, private paths, or machine-specific credentials are included
- [ ] `.agent/skills/README.md` updated
- [ ] Category README updated
- [ ] Repository root `README.md` still matches the actual structure
